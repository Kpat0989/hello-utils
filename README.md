# hello-utils

A small collection of Python utility functions.

## Functions

- `greet(name)` — returns a personalised greeting string
- `add(a, b)` — returns the sum of two numbers

## Usage

```python
from utils import greet, add

print(greet("world"))   # Hello, world!
print(add(2, 3))        # 5
```

## Running tests

```bash
python -m pytest
```
